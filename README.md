# One-Piece-Durak ⚓

Kartenspiel nach Durak-Regeln mit offiziellen One-Piece-Sammelkarten (Bandai OPTCG).

**Spielen:** https://nickillig3-dotcom.github.io/one-piece-durak/

- 🌐 **Online-1v1**: Raum erstellen, Code/Link an den Kumpel schicken, losspielen — jeder auf seinem Handy (WebRTC-Direktverbindung, keine Anmeldung).
- 👥 **Zu zweit an einem Handy** (Hot-Seat mit Sichtschutz).
- Verteidigung nur mit klar stärkerer Karte (Kanon-Stärke), Nachlegen nur mit Crew- oder Kanon-Kampf-Verbindung, max. 5 Angriffskarten — inkl. eingebautem Schiedsrichter, der jede Entscheidung begründet.
